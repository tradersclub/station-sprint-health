# Challenge 2
Recebendo uma lista de 1.000.000 usuários que possui um ID e um NAME, crie uma estrutura onde a busca, criação e deleção demore menos que 1ms para ser realizada.

Exemplo de input:
```ts
// Typescript
type user = {
    id: number;
    name: string;
}
```

```go
// Golang
type user struct {
    id int
    name string
}
```
